* アクセス修飾子
・TOP LEVEL（クラス）
public            -> どこからでもアクセス可能
(package private) -> 同じパッケージからのみアクセス可能

・フィールド，メソッド，etc...
public            -> どこからでもアクセス可能
(package private) -> 同じパッケージからのみアクセス可能
private           -> 同じクラスからのみアクセス可能
protected         -> 同じパッケージからアクセス可能（別パッケージであってもサブクラスからならアクセス可能）

・static修飾子
static            -> インスタンス化しなくてもクラスから直接扱えるフィールドやメソッドを定義することが可能

・static修飾子
final            -> finalをつけたクラスやフィールドやメソッドは変更不可
